Project Flow

Receive incoming email from the inbox

Extract email subject, body, and attachments

Check whether the attachment is a PDF file

Extract data from the PDF using pdfplumber

Convert extracted data into structured Excel format

Generate an .xlsx file

Send the Excel file back to the sender via email using SMTP
